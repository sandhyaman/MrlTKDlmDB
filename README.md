# Python图书信息管理系统

## 核心功能
- 登录注册
- 书籍信息管理
- 书籍排序
- 书籍搜索
- 出版社信息管理
- 作者信息管理
- 分页查询

## 适用场景
- 图书馆信息管理
- 书店库存管理
- 教育机构图书资源管理

## 技术特点
- 使用Django框架开发，确保系统稳定可靠
- 数据库采用MySQL，保证数据存储安全
- 前端界面采用HTML、CSS、JS，界面友好，操作便捷
